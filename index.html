Project on Console

Visual Studio Code is an app that lets you code in all different languages. In Order to use Visual Studio Code, you need to make a folder and input coding languages to start.

The Console screen is used to print stuff and, it is used to debug your code to find any errors you made in order to have a proper page. To open the Console screen you need to right click and press inspect. Just click on the console button and you can check your code.
